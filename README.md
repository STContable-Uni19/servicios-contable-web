✨ Soporte Técnico Contable Unificado

Bienvenido al repositorio oficial de Soporte Técnico Contable Unificado, una plataforma integral diseñada para ofrecer soluciones contables y jurídicas de alta gama en Argentina.

🏆 Nuestra Propuesta de Valor

Combinamos la precisión de la técnica contable con la solidez del derecho para brindar un asesoramiento premium y personalizado.

🚀 Servicios Destacados

Devolución de Percepciones (RG 4815): Recupero del 35% por compras en el exterior, Netflix, Spotify, Steam y viajes.

Actuación Jurídico-Contable: Especialistas en procesos de Concursos y Quiebras.

Gestión de Monotributo: Altas, recategorizaciones y optimización fiscal.

Asesoría para Profesionales: Soporte específico para Abogados y Escribanos.

🤖 Innovación con IA

La página integra un Asistente Fiscal Inteligente (basado en Gemini 2.5 Flash) que permite a los usuarios:

Simular el recupero de sus percepciones.

Analizar casos complejos de deuda.

Obtener una respuesta preliminar antes de la consulta personalizada.

🛠️ Tecnologías Utilizadas

Frontend: HTML5, Tailwind CSS (Diseño Luxury Dark/Gold).

Interactividad: JavaScript ES6.

Inteligencia Artificial: Google Gemini API.

Despliegue: GitHub Pages.
